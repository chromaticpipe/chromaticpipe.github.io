SRB2 CROSS COMPILING GUIDE

This is a short guide for cross compiling SRB2 for multiple platforms, like for example, compiling a Windows version of SRB2 on a GNU/Linux system.

Let's get straight to the point

Part One: Getting SRB2 Source


Of course, to compile SRB2, you need a copy of the source code
You will need Git, a source control software that SRB2's source is hosted on
To get Git, visit Git's official website and find the install instructions for your distro
Once you did that, you can get SRB2's source with the following command

git clone https://git.do.srb2.org/STJr/SRB2.git

Part Two: Obtaining Toolchains


To cross compile for different systems, you need different toolchains(which are collections of software that include all the nessecary programs, like compilers, linkers, etc.)
This guide will be showing you how to install the MinGW toolchain, which allows you to compile and cross compile for Windows systems
Below are the install commands for each distro

Debian/Ubuntu/Linux Mint/Pop! OS

sudo apt install mingw-w64

Arch/Arch Based

pacman -S mingw-w64-gcc

Fedora/Fedora Based

sudo dnf install mingw32-gcc mingw64-gcc
